# AI-Powered Credit Card Fraud Detection Dashboard

## 📌 Project Overview
This project is a machine learning-powered fraud detection system with an interactive web dashboard built using FastAPI and Chart.js.

## 🚀 Features
- Batch-wise fraud prediction (284,807 transactions supported)
- Animated dashboard UI
- Fraud vs Legitimate comparison
- Fraud trend over time
- Feature importance visualization
- Downloadable prediction results

## 🛠 Tech Stack
- Python
- FastAPI
- Scikit-learn
- Pandas
- Chart.js
- Bootstrap

## ▶ How to Run

1. Create virtual environment
2. Install requirements
   pip install -r requirements.txt
3. Train model
   python train_model.py
4. Run server
   uvicorn app:app --reload

Open:
http://127.0.0.1:8000

## 📊 Dataset
Credit Card Fraud Detection Dataset (Kaggle)
